## Setup
### Install
```bash
# Clone the repository
git clone git@github.com:untitled-dream/react-mesto-auth.git

# Enter the project directory
cd react-mesto-auth

# Install dependencies
npm i
```
### Scripts
```bash
# Run a local server at localhost:3000
npm start
