## Setup
### Install
```bash
# Clone the repository
git clone git@github.com:untitled-dream/mesto-react.git

# Enter the project directory
cd mesto-react

# Install dependencies
npm i
```
### Scripts
```bash
# Run a local server at localhost:3000
npm start
